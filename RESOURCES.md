# 🛠️ Quality Engineering: Essential Toolkit & Resources
### *Curated assets to enhance testing efficiency, data integrity, and evidence collection.*

To deliver high-quality results in fast-paced environments, I leverage a specialized set of tools that streamline execution and ensure technical precision.

---

## 📹 Evidence Collection & Reporting
*Capturing clear, reproducible steps is non-negotiable for high-impact bug reporting.*

* **[Loom](https://www.loom.com/):** Ideal for quick video bug reports with voiceover. Perfect for explaining complex UX issues to developers.
* **[ScreenToGif](https://www.screentogif.com/):** A powerful, open-source tool to record screen areas and save them as lightweight GIFs. Excellent for JIRA attachments.
* **[Lightshot](https://app.prntscr.com/):** The fastest way to take customizable screenshots with instant annotation tools.
* **[X-Ray Exploratory App](https://www.getxray.app/exploratory-testing-app):** A desktop tool to help document exploratory sessions with screenshots and notes that sync directly with Test Management tools.

---

## 💾 Test Data & Integrity
*Validating the "truth" behind the UI requires controlled and realistic datasets.*

* **[Mockaroo](https://www.mockaroo.com/):** My primary tool for **Synthetic Data Generation**. It generates thousands of rows in CSV, JSON, or SQL formats with realistic patterns (Names, SSNs, credit cards).
* **[JSONPlaceholder](https://jsonplaceholder.typicode.com/):** A free online REST API for testing and prototyping when the backend is still under development.
* **[Faker (Library)](https://fakerjs.dev/):** For automated scripts, I use Faker to generate massive amounts of fake data on the fly.
* **[SQL Fiddle](http://sqlfiddle.com/):** A "sandbox" to test and share SQL queries and database schemas during the Data Integrity validation phase.

---

## 🦾 Accessibility (A11y) & UX
*Ensuring the product is inclusive and follows WCAG 2.1/2.2 standards.*

* **[Axe DevTools](https://www.deque.com/axe/):** The industry standard for automated accessibility audits within the browser.
* **[WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/):** Critical for validating color contrast ratios against AA/AAA standards.
* **[WAVE Evaluation Tool](https://wave.webaim.org/):** A visual community tool that provides feedback on web content accessibility by injecting icons into the page.
